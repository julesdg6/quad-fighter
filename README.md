# Quad Fighter Prototype

## Requirements

To run the game, install the required dependencies:

```bash
pip install -r requirements.txt
```

## Controls

- Arrow keys: Move left/right (lane change for depth)
- Space: Jump
- A: Attack

## Running the Game

```bash
python main.py
```

## Features Implemented

- Player movement with lane-based depth
- Basic enemy AI (chase player)
- Simple combat system with attack collision detection
- Knockback mechanics
- Basic hitbox visualization
- 60 FPS game loop
